Nederlandse MicroPayment plugin voor WordPress.

== Description ==
De content op uw website is waardevol voor u, maar ook voor uw bezoeker. Vergroot binnen vijf minuten de omzet van uw website door uw content op eenvoudige wijze betaald aan te bieden!

PayPress is een MicroPayment PayWall plugin voor WordPress. Met deze eenvoudig te gebruiken plugin kunt u bezoekers per telefoon laten betalen voor de waardevolle content die op uw website staat. U geeft zelf aan voor welke pagina's of berichten betaald moet worden en de bezoeker rekent makkelijk met u af per telefoon.

Met PayPress kunnen bezoekers betalen via Pay-Per-Call waarmee ze door 1 keer bellen 1,30 afrekenen en direct toegang krijgen tot uw content.

== Installation ==

1. Maak een PayPress account aan op my.paypress.nl
2. Upload de content uit paypress.zip naar de map '/wp-content/plugins/'
3. Activeer PayPress in het 'Plugins' menu in WordPress
4. Geef uw persoonlijke activatiecode op in de beheeromgeving van de plugin.
5. Verkoop direct de content op uw website!

== Frequently Asked Questions ==

= Wat is PayPress =
Laat bezoekers betalen voor jou WordPress content met een telefoontje. Binnen 5 minuten zet jij betaalmogelijkheden op jouw WordPress website of weblog. Vermarkt jouw waardevolle content!

= Voor wie is PayPress =
PayPress is voor iedereen die de content op zijn WordPress website betaald wil aanbieden via de telefoon. 

= Is PayPress verantwoordelijk voor uw content? =
Nee, u bepaalt zelf welke content u betaald aanbiedt. PayPress is slechts de toegangspoort tot uw content. De content is van uzelf en niet van PayPress.

= Kan iedereen gebruik maken van PayPress? =
Nee. Deze versie van PayPress is alleen geschikt voor micropayments via Nederlandse telefoons. 

= Hoeveel krijg ik uitbetaald van PayPress? =
Kijk voor de actuele nettobedragen op www.paypress.nl
